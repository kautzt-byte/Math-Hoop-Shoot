# Video Clips

Put short web-friendly video clips here.

Notes:
- HEVC (H.265) is not supported in all browsers. For best compatibility on GitHub Pages, prefer an H.264 `.mp4` (and optionally a `.webm` fallback).
- `.mov` files are often not supported in browsers. For GitHub Pages, export to H.264 `.mp4` (recommended).
- Current code expects the Landyn clip at `assets/video/landyn.mp4`.
